Foco em interface do usuario, UX e integracao completa.

Entregas Tecnicas:

Single Page Application (SPA) desenvolvida com React e Vite.

Estilizacao moderna utilizando Tailwind CSS v4.

Gerenciamento de estado para feedback instantaneo (Toasts) e atualizacao automatica de tabelas.

Finalizacao da observabilidade com Spring Actuator e Swagger UI.
